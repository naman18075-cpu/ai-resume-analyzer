# AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes against job descriptions using ATS-style scoring and intelligent feedback.

## Features

- Resume upload
- ATS score generation
- Job description matching
- Dashboard analytics
- Authentication system
- AI-powered suggestions

## Tech Stack

Frontend:
- React
- Vite
- TailwindCSS

Backend:
- FastAPI
- PostgreSQL
- JWT Authentication

## Setup

Frontend:
npm install
npm run dev

Backend:
pip install -r requirements.txt
uvicorn app.main:app --reload
